What is CSS?


  1. CSS stands for Cascading Style sheets

  2. CSS describes how HTML elements are to be displayed on screen,paper or in other media

  3. CSS is not neccesary CSS saves a lot of work. It can control the layout of multiple web pages all at once

  4. External stylesheets are stored in CSS files

CSS can be added to HTML documents in 3 ways:

  1. Inline - by using the style attribute inside HTML elements

  2. Internal - by using a style element in the head section

  3. External - by using a link element to link an external CSS file.

  4. The most common wat to add CSS, is to keep the styles in external CSS files.However,in this tutorial we will use inline and internal styles, because this is easier to demonstrate and eaasier for you to try it yourself.

When comparing CSS class vs Id, the difference in that CSS class applies a style to multiple elements, ID, on the other handm applies a style to one unique element.ID is also special in that you can use a special URL to link directly to an element and it's used in JavaScript.

In CSS, selectors are used to target a specific elment of range of elements on a web page. Once an element has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors avaulable.Two os most commonly used are class and ID. Both are used to target elements to which aa style should be applied.
